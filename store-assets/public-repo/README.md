## What is VideoLM?

VideoLM bridges the gap between YouTube and Google NotebookLM. Instead of manually copying video URLs one by one, VideoLM adds a native button directly on YouTube pages — click it to import videos, entire channels, or full playlists into your notebook instantly.

## Key Features

| Feature | Description |
|---|---|
| **One-Click Import** | Click the NotebookLM button on any YouTube video page — no tab switching needed |
| **Batch Import** | Import all videos from a channel, playlist, or search results in a single click |
| **Smart Chunking** | Automatically splits 100+ video collections into Part 1, Part 2... notebooks (50-source limit solution) |
| **Smart Dedup** | Checks existing notebooks and skips already-imported videos. Supports CJK name matching |
| **Auto-Create** | Automatically creates and names notebooks after the channel or playlist |
| **Real-Time Progress** | Shadow DOM toast notifications show live import progress without interfering with YouTube |

## How It Works

1. Install VideoLM from the Chrome Web Store
2. Browse YouTube as you normally do
3. Click the VideoLM button on any video, channel, or playlist page
4. VideoLM sends the content directly to NotebookLM
5. Open NotebookLM to chat with your videos, generate summaries, or create Audio Overviews

## Use Cases

- **Researchers** — Build comprehensive video knowledge bases on any topic
- **Students** — Import lecture playlists for AI-powered study guides
- **Podcasters** — Curate source videos and generate Audio Overviews
- **Content Creators** — Research competitors, analyze trends, organize inspiration

## Supported Pages

- YouTube video pages (`/watch`)
- YouTube channel pages (`/@handle`)
- YouTube playlist pages (`/playlist`)
- YouTube search results (`/results`)
