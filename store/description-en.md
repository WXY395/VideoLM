VideoLM — AI Video to NotebookLM

Transform YouTube videos into structured, AI-enhanced knowledge sources for Google NotebookLM.

Features:
- One-click import from YouTube to NotebookLM
- AI-powered structured summaries with outlines, key points, and timestamps
- Smart chapter splitting — each chapter becomes a separate Source for precise retrieval
- Multi-language translation before import
- Duplicate detection to keep your notebooks clean
- BYOK (Bring Your Own Key) — use your own OpenAI or Claude API key
- Privacy-first: API keys stored locally, never sent to our servers

How It Works:
1. Open any YouTube video
2. Click VideoLM icon
3. Choose import mode (Raw / AI Structured / AI Summary / Chapter Split)
4. Content is processed and imported into your NotebookLM notebook
